# VyomGarud Landing Page

A modern, responsive landing page for **VyomGarud**, a UAV/drone systems company with a military-grade, professional identity.


## Tech Stack

- Next.js 14 (React framework with App Router)  
- Tailwind CSS for styling  


## Deployment

Deployed live on Vercel:  
[https://vyom-garud-kiran.vercel.app/](https://vyom-garud-kiran.vercel.app/)


## Commands

```bash
# Create new Next.js project (JavaScript)
npx create-next-app@latest vyomgarud --javascript

# Install dependencies
npm install

# Run development server
npm run dev
